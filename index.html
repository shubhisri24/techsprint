Instant Restoration

Point your device at any heritage site. Our ARCore engine overlays hyper-realistic 3D reconstructions with centimeter-level precision. See temples as they stood 1,000 years ago, not as they look today.

AI-Assisted Audio Storytelling

Scan an inscription, plaque, or monument marker to unlock narrated historical context. Using AI-powered speech synthesis built into modern browsers, the system reads out curated stories in a clear, natural voice.

History is too cool to be boring.

The Google-Powered Stack

WebXR & WebGL Foundations

Our experience is built on modern WebXR and WebGL standards, implemented and led by Google through the Chrome platform. This enables camera-based, AR-like 3D experiences on mobile devices using only the browser.

Interactive Mapping & Route Intelligence

The platform is built on Leaflet.js, a lightweight open-source mapping library, to visualize Delhi’s heritage monuments on an interactive map. Using Leaflet Routing Machine, users can generate routes between monuments, enabling walking or travel-based heritage trails.

AI-Powered Voice Narration

We integrate Google’s Web Speech API to provide AI-driven text-to-speech narration. Monument stories are read aloud using natural-sounding voices, improving accessibility and engagement. This uses machine-learning speech synthesis built into Google Chrome — no external servers or exposed API keys.

Google Chrome Platform Capabilities

The project relies on core Google Chrome technologies including secure camera access, on-device machine learning, WebGL rendering, and HTTPS security. These capabilities allow QR-to-3D workflows and smooth real-time interaction.

Documentation is the only way to save the past.

Every year, physical heritage sites are lost to climate, urban sprawl, and time. Heirs of History creates a permanent, digital “living record.” By making heritage interactive and social-ready, we ensure the next generation values preservation as much as progress.
